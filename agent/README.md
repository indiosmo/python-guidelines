# Agent Context

This folder contains the concise agent-facing layer for the Python guideline
set.

- [python-agent-context.md](python-agent-context.md) gives always-loaded rules
  for agents editing Python code.
- [python-agent-examples.md](python-agent-examples.md) gives good and bad
  Python examples keyed to the context sections.

Load `python-agent-context.md` unconditionally in a consuming repository. Load
`python-agent-examples.md` when an edit or review needs concrete examples.
